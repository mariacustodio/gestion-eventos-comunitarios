# Aplicación de Gestión de Eventos Comunitarios

## Descripción
Esta aplicación móvil está desarrollada en **Android con Kotlin** y permite a una comunidad local gestionar eventos y actividades comunitarias. Los usuarios pueden registrarse, iniciar sesión, crear eventos, confirmar asistencia, dejar comentarios y ver el historial de eventos pasados. La aplicación utiliza **Firebase** para autenticación y almacenamiento de eventos.

## Funcionalidades de la Aplicación
1. **Autenticación**
    - Registro y acceso mediante correo y contraseña.
    - Integración con Firebase Authentication.

2. **Gestión de Eventos**
    - Crear eventos con título, descripción, fecha, hora y ubicación.
    - Visualizar lista de eventos próximos.
    - Confirmar asistencia a eventos.

3. **Interacción Social**
    - Visualización de detalles del evento.
    - Comentarios y calificaciones (pendiente de implementar).

4. **Historial**
    - Registro de eventos pasados y asistencia del usuario.
